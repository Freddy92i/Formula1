# F1
Site projet codé principalement en langauage Front-End est un site présentant une sorte de Blog F1 lié a une API.

## Présentation
Ce projet est une sorte de blog avec toutes les dernières informations liée a la Formule 1.
liée a une API, le site vous montreras des nombreuse statistiques présente sur l'API Ergast F1.

Ce site permet de voir :

> - Le championnat constructeur de la saison actuelle.
> - Le championnat pilote de la saison actuelle.
> - Les archives de la Formule 1 - Les championnats Pilotes et Constructeurs depuis la création de la Formule 1 (1950).
> - Les résultats des week-end de grand prix de la saison actuelle ( gp de la saison avec le podium et les intervalles entre pilotes).
> - Liste des pilotes de la saison actuelles.
> - Liste des Constructeurs de la saison actuelles.
> - Le Calendrier de la saison actuelle.
> - Les dernieres actualités dans le monde de la F1 ( Feed RSS - Motorsport.com/F1)

## Contenu

Site utilisant différents languages tel que

> JS
> HTML
> CSS
> PHP ( pour pouvoir afficher le feed rss car était bloqué par la méthode CORS en JS)

## Comment l'installer ?

Télécharger le projet,
mettre le projet dans votre serveur web
y modifier si besoin la clé api vers Ergast F1
lancer le projet en lançant l'index.php.
